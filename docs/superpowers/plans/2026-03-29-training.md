# Training (Capability 5) Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Implement the full training pipeline — backward passes for all layers, CrossEntropyLoss, AdamW, gradient clipping, LR schedule, training loop, metrics, and binary checkpoint save/load.

**Architecture:** Manual chain-rule backprop (no autograd). A `Parameter` class (in Core) wraps each weight Tensor + its gradient Tensor. All inner layers implement `ILayer` (also in Core). The `PicoLLM.Training` project builds on Core and adds the optimizer, scheduler, training loop, and checkpoint manager.

**Tech Stack:** C# 12 / .NET 9, PicoLLM.Core (zero deps), xUnit + FluentAssertions.

---

## Key Design Decisions

1. `Parameter` and `ILayer` live in **PicoLLM.Core** (avoids circular deps — layers are in Core and must implement ILayer).
2. Every layer caches its forward-pass inputs for use in Backward().
3. `Parameters()` return type changes from `IEnumerable<Tensor>` to `IEnumerable<Parameter>` across all layers. Existing public properties (`Weights`, `Bias`, `Gamma`, `Beta`) remain as pass-throughs to `param.Data` for backward compat.
4. `GetAllParameters()` on `PicoLLMModel` changes to `IReadOnlyList<Parameter>`.
5. One existing test needs updating: `GetAllParameters_SumMatchesTotalParameters` — `.Sum(p => p.Length)` becomes `.Sum(p => p.Data.Length)`.
6. `TensorFactory.FromArray(int[] shape, float[] data)` added for Training's tensor construction.

## Backward Pass Formulae

**LinearLayer** (`y = x @ W + b`, shapes `[rows,in] @ [in,out] → [rows,out]`):
- `dW += x^T @ dL/dy`
- `db += Σ(dL/dy, axis=0)`
- `dx = dL/dy @ W^T`

**LayerNorm** (`y = γ * x̂ + β`, where `x̂ = (x-μ)/σ`):
- `dγ += Σ(dL/dy * x̂)`
- `dβ += Σ(dL/dy)`
- `dx = (1/D) * invStd * [D*dL/dy·γ − Σ(dL/dy·γ) − x̂*Σ(dL/dy·γ·x̂)]`

**MultiHeadAttention** (caches: `_lastInput`, `_qHeads`, `_kHeads`, `_vHeads`, `_attnWeights`):
1. OutputProj.Backward → dContext [B,S,E]
2. Permute/reshape dContext → [B,H,S,D]
3. `dV = attnWeights^T @ dContext`; `dWeights = dContext @ vHeads^T`
4. SoftmaxBackward: `dScores[i] = attn[i] * (dWeights[i] - Σ_j(dWeights[j]*attn[j]))`
5. `dQ = dScores @ kHeads / √D`; `dK = dScores^T @ qHeads / √D`
6. Un-permute/reshape dQ,dK,dV → [B,S,E]; project backward through Q/K/V projections; sum three dx

**FeedForward** (caches: `_upOutput` = pre-GELU values):
- down.Backward → dHidden; `dUpOutput = dHidden * GELU'(_upOutput)`; up.Backward(dUpOutput)

**DecoderBlock** (pre-norm residual, reverse order):
- FFN path: `dY1 = gradOut + _ffnNorm.Backward(FFN.Backward(gradOut))`
- Attn path: `dX = dY1 + _attnNorm.Backward(Attn.Backward(dY1))`

**CrossEntropyLoss**:
- Forward: `loss = -mean(log(softmax(logits)[b,s,target[b,s]]))`
- Backward: `dLogits = softmax(logits); dLogits[b,s,target[b,s]] -= 1; dLogits /= (B*S)`

**AdamW** (per parameter, step t):
- `m = β1*m + (1-β1)*g`; `v = β2*v + (1-β2)*g²`
- `m̂ = m/(1-β1^t)`; `v̂ = v/(1-β2^t)`
- `p -= lr * (m̂/(√v̂+ε) + weightDecay*p)`

**GradientClipper**:
- `globalNorm = √Σ||g||²`; if `globalNorm > maxNorm`: `g *= maxNorm/globalNorm`

**LRSchedule** (warmup+cosine):
- step < warmupSteps: `lr = maxLr * step/warmupSteps`
- else: `lr = minLr + 0.5*(maxLr-minLr)*(1 + cos(π*(step-warmupSteps)/(totalSteps-warmupSteps)))`
